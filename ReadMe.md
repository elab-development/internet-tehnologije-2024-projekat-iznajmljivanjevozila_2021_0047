\# VoziloNaDan Aplikacija



Ovo je aplikacija za iznajmljivanje vozila. Projekat sadrži dva dela:

\- \*\*Backend\*\*: Laravel API

\- \*\*Frontend\*\*: React aplikacija



\## Struktura projekta

projekat/

├── backend/ # Laravel API

├── frontend/ # React aplikacija

└── README.md 

\## Backend (Laravel)

\*\*Tehnologije:\*\* PHP 8, Laravel 11, MySQL  



\### Instalacija

Backend

composer install

php artisan migrate

php artisan serve

Frontend

npm install

npm start
